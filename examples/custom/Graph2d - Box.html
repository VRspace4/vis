Graph2d | Bar Graph Example

fff example shows the most the same data as the first example, except we plot the data as bars! The dataAxis (y-axis) icons have been enabled as well. These icons are generated automatically from the CSS styling of the graphs. Finally, we've used the option from Timeline where we draw the x-axis (time-axis) on top.

The align option can be used to align the bar at the center of the datapoint or on the left or right side of it. This example uses the default center alignment.
